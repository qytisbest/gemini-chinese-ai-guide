## 机场选择标准

选择机场时，不建议只看价格，更应该关注以下几个方面：

| 评测维度 | 说明 |
|---|---|
| 线路质量 | 是否支持专线、中转优化、低延迟节点 |
| 晚高峰稳定性 | 晚上 8 点到 12 点是否稳定 |
| 节点覆盖 | 是否覆盖香港、日本、新加坡、美国等常用地区 |
| AI 工具支持 | 是否支持 ChatGPT、Gemini、Claude、Copilot |
| 流媒体解锁 | 是否支持 Netflix、Disney+、YouTube、TikTok |
| 套餐价格 | 月付、季付、年付价格是否合理 |
| 客服响应 | 出现问题是否能及时处理 |
| 新手友好度 | 是否有教程、官方客户端或简单配置方式 |

## 新手怎么选机场？

如果你是第一次购买机场，建议优先按照下面方式选择：

1. 先选择月付或季付套餐测试
2. 重点测试晚高峰速度
3. 测试常用节点是否稳定
4. 测试 ChatGPT、Gemini、TikTok、Netflix 是否可用
5. 满意后再考虑年付或长期套餐

不建议新手一开始直接购买多年套餐，应该先确认机场的稳定性和使用体验。
